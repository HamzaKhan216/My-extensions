## 🚀 Overview

**Omni-Copilot AI** is a lightweight, high-performance Chrome Extension that brings the power of state-of-the-art Large Language Models directly to your browser's side panel. Read, summarize, and analyze the webpage you are currently viewing instantly without leaving the tab. 

Designed for peak productivity without the bloat, Omni-Copilot allows you to seamlessly switch between the world's most powerful AI models, including **ChatGPT (OpenAI)**, **Claude (Anthropic)**, **Google Gemini**, **Groq**, and **Nvidia NIM**.

## 🌟 Key Features

* **🌐 Universal Webpage Context:** Omni-Copilot can intelligently read the active tab. Highlight specific text, or let our smart DOM extractor bypass noisy headers and footers to fetch the core content of the page for the AI to analyze.
* **🧠 Multi-Model Support:** Plug in your API keys and instantly use:
  * `OpenAI` (GPT-4o)
  * `Anthropic` (Claude 3 Opus/Sonnet/Haiku)
  * `Google` (Gemini 1.5 Pro)
  * `Groq` (Llama 3, Mixtral)
  * `Nvidia NIM` (Meta Llama 3 70b)
* **📸 Multimodal Capabilities:** Paste images directly into the chat prompt to have vision-capable models analyze them alongside your text.
* **⚡ Keyboard Shortcuts & Quick Actions:** Hit `Ctrl+Shift+E` (or `Cmd+Shift+E`) to instantly open the side panel anywhere. Use one-click buttons to summarize or extract key takeaways.
* **📝 Custom System Prompts:** Tailor the AI's persona and output formatting natively through the settings panel to fit your unique workflow.
* **📚 Secure Local Chat History:** All your chats and API keys are stored securely in your browser's local storage. Revisit old sessions with perfectly preserved Markdown rendering and beautiful syntax highlighting.

## 🛠️ Installation

1. Clone or download this repository to your local machine:
   ```bash
   git clone https://github.com/your-username/omni-copilot-extension.git
   ```
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click on **Load unpacked** and select the directory where you cloned this repository.
5. Pin the extension to your toolbar.
6. Open the side panel, click the ⚙️ **Settings** gear, and enter your preferred provider's API key!

## ⌨️ Shortcuts

- **Open Omni-Copilot Side Panel:** `Ctrl + Shift + E` (Windows/Linux) or `Cmd + Shift + E` (Mac).
- **Send Message:** `Enter`
- **New Line:** `Shift + Enter`

## 🎨 UI/UX Highlights
- **Beautiful Code Blocks:** Inspired by premium developer aesthetics, featuring vibrant syntax highlighting and a 1-click copy button.
- **Floating Scroll-to-Bottom Button:** A sleek glassmorphism button that automatically appears when you scroll up during long conversations.
- **Thinking Blocks:** Native support for visualizing "Thinking Processes" generated by models (like Groq/Llama) inside collapsible UI sections.

## 🔒 Privacy & Security
Omni-Copilot operates entirely locally. Your API keys, chat history, and system prompts are saved strictly to `chrome.storage.local`. The extension does not collect analytics, telemetry, or user data. It only communicates with the specific AI API endpoint you configure.
